FAQ

Frequently asked questions

Is my monitor compatible?
Lumina Control works with all monitors that support the DDC-CI protocol (the vast majority of modern desktop screens). Make sure DDC-CI is enabled in your monitor's OSD menu. Laptop built-in panels don't support DDC-CI, but Lumina Control controls them via the WMI backend for brightness. For all screens, the tone curve editor and GPU gamma work without DDC-CI.
Can I use it on multiple PCs?
The Pro licence allows installation on 2 machines simultaneously. If you change PC, contact support for a reactivation.
Does it work with HDR?
DDC-CI control is disabled by Windows when HDR is active on a monitor. Lumina Control then uses the GDI32 gamma LUT for tint and gamma adjustments, but DDC-CI brightness won't be available while HDR is on.
Is there a free trial?
No trial at the moment. If the software doesn't work correctly on your configuration within 7 days of purchase, contact support for a full refund.
What's the difference from Windows Night Light?
Windows Night Light is a software overlay that tints the screen orange. Lumina Control acts directly on the monitor firmware via DDC-CI: physical brightness actually decreases, reducing eye strain and power consumption. No overlay, no color degradation.
Does the curve editor work without DDC-CI?
Yes — tone curves and GPU gamma (SetDeviceGammaRamp) go through GDI32 and work on all screens, even without DDC-CI. ICC v2 profile export is also independent of DDC-CI. Only the R/G/B gains in the first tab require an active DDC-CI connection.
Can I import the ICC profile into Photoshop / Lightroom?
Yes. The ICC v2 profile generated by Lumina Control encodes D50-adapted sRGB primaries and your correction curves in the TRC tags. Import it into Photoshop (Edit → Color Settings) or set it as your screen profile in Windows settings so DaVinci Resolve, Lightroom and Premiere Pro recognize it automatically.
