FAQ

Questions fréquentes

Mon moniteur est-il compatible ?
Lumina Control fonctionne avec tous les moniteurs qui supportent le protocole DDC-CI (la quasi-totalité des écrans de bureau modernes). Assurez-vous que DDC-CI est activé dans le menu OSD de votre moniteur. Les dalles internes de laptops ne supportent pas DDC-CI, mais Lumina Control les contrôle via le backend WMI (Windows Management Instrumentation) pour la luminosité. Pour tous les écrans, l'éditeur de courbes tonales et le gamma GPU fonctionnent aussi sans DDC-CI.
Puis-je l'utiliser sur plusieurs PC ?
La licence Pro autorise l'installation sur 2 machines simultanément. Si vous changez de PC, contactez le support pour une réactivation.
Ça fonctionne avec HDR ?
Le contrôle DDC-CI est désactivé par Windows lorsque HDR est actif sur un moniteur. Lumina Control utilise alors la LUT gamma GDI32 pour les ajustements de teinte et gamma, mais la luminosité DDC-CI ne sera pas disponible tant que HDR est activé.
Est-ce qu'il y a un essai gratuit ?
Pas d'essai pour le moment. Si le logiciel ne fonctionne pas correctement sur votre configuration dans les 7 jours suivant l'achat, contactez le support pour un remboursement complet.
Quelle différence avec le mode nuit de Windows ?
Le mode nuit de Windows est une surcouche logicielle qui teinte l'écran en orange. Lumina Control agit directement sur le firmware du moniteur via DDC-CI : la luminosité physique baisse réellement, ce qui réduit la fatigue oculaire et la consommation électrique. Aucune surcouche, aucune dégradation des couleurs.
L'éditeur de courbes fonctionne-t-il sans DDC-CI ?
Oui — les courbes tonales et le gamma GPU (SetDeviceGammaRamp) passent par GDI32 et fonctionnent sur tous les écrans, même sans DDC-CI. L'export profil ICC v2 est également indépendant du DDC-CI. Seuls les gains R/V/B du premier onglet nécessitent une connexion DDC-CI active.
Puis-je importer le profil ICC dans Photoshop / Lightroom ?
Oui. Le profil ICC v2 généré par Lumina Control encode les primaires sRGB adaptées au point blanc D50 et vos courbes de correction dans les tags TRC. Importez-le dans Photoshop (Édition → Couleurs) ou définissez-le comme profil de l'écran dans les paramètres Windows pour que DaVinci Resolve, Lightroom et Premiere Pro le reconnaissent automatiquement.
